# Cyclistic Bike-Share Analysis

This repository contains a data analysis case study based on Cyclistic bike-share usage data.

The analysis focuses on identifying behavioural differences between casual riders and annual members, with the goal of supporting data-driven marketing and membership conversion strategies.
